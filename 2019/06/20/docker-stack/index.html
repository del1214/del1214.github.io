docker stack

docker stack 需要在编写完 docker-compose(会读取Compose配置文件) 和创建好 docker swarm(使用Node Worker) 后使用

docker stack deploy

开启stack，更新stack
别名

docker stack up

部署docker-compose编排服务

docker stack rm

停止stack，删除stack
别名

docker stack down

docker stack ls

显示stack列表

别名

docker stack list

docker stack ps

显示正在运行的stack列表

docker stack services

显示stack中的服务

读取日志
